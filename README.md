# predictmotion
HyperOdds - HyperMotion League Probability Analyzer
A comprehensive web application that calculates and displays real-time winning probabilities for each team in the Spanish HyperMotion League (Segunda División B). This platform provides data-driven insights into team performance using advanced statistical analysis.
Features:

Real-time probability calculations based on team performance metrics
Interactive visualization of odds for all HyperMotion teams
Responsive design optimized for desktop and mobile devices
Clean, professional user interface for easy navigation
Privacy policy and terms of service included
Google AdSense monetization integration

Tech Stack:

HTML5 / CSS3 / JavaScript (vanilla)
Static site hosted on Netlify for 24/7 availability
Google AdSense for revenue generation

Project Structure:

index.html — Main application page with probability dashboard
privacy.html — Privacy policy and legal compliance
ads.txt — AdSense configuration file
logos/ — Team logos and branding assets

How It Works:
The application analyzes historical performance data and current standings from the HyperMotion League to compute mathematical probabilities of each team winning the championship or achieving specific goals. Results are displayed in an easy-to-understand format with visual indicators.
Deployment:
Currently hosted on Netlify with automatic HTTPS and global CDN distribution. The site is live 24/7 without requiring local server infrastructure.
